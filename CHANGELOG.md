# Changelog

Все заметные изменения в проекте MEMORIAL будут фиксироваться в этом файле.

## [1.0.0] - 2026-03-11

### Добавлено

- **Агентская архитектура (Antigravity Skills):**
  - Создан навык `memorial-analyzer` для автоматического анализа исходных фото и заполнения данных заказа.
  - Создан навык `memorial-prompter` для динамической сборки промптов из атомарных блоков.
  - Создан интерактивный чек-лист `memorial-postprocessing` для финальной ретуши в Photoshop.
- **Управление заказами:**
  - Внедрена папочная структура для заказов (`orders/active/`, `orders/archive/`).
  - Разработана JSON-схема (`orders/schema.json`) для стандартизации данных.
  - Создан шаблон заказа и тестовый пример (`TEST_ORDER`).
- **База знаний (Knowledge Base):**
  - Описаны технические требования для [Лазерных](knowledge/machines/laser.md) и [Ударных](knowledge/machines/impact.md) станков.
  - Сформулированы фундаментальные принципы ретуши в `knowledge/principles.md`.
- **Библиотека промптов:**
  - Созданы атомарные блоки для базовых инструкций, станков, типов одежды (военная/гражданская) и головных уборов.
- **Навигация:**
  - Полностью переработан `workflow.md` как центральный путеводитель по системе.

### Изменено

- **README.md:** Полностью обновлен для отражения новой агентской структуры и принципов работы в Antigravity IDE.
- **Workflow:** Процесс переведен с линейного чтения документации на итеративное взаимодействие с агентами (Skills).

### Удалено

- Устаревшие монолитные файлы документации: `pipeline_laser.md`, `pipeline_impact.md`, `правки портрета.txt`.

## [0.1.0] - 2026-03-10

- Начальная версия проекта с базовой документацией по пайплайнам.
