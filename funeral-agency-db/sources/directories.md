# База источников для поиска ритуальных агентств

## Источники (в порядке приоритета)

### Карты

| № | Источник | URL-шаблон | Описание |
|---|----------|------------|----------|
| 1 | 2GIS | `https://{город}.2gis.ru/search/ритуальные+услуги` | Карты, отзывы, фото |

## Поисковые системы

### Поиск организаций

```
ритуальные услуги памятники {город}
изготовление памятников {город}
надгробные памятники {город}
гранитные памятники {город}
```

### Поиск Telegram по номеру

```
"{номер}" Telegram
```

Примеры:
- `+7 903 955 81 17 Telegram`
- `8 937 821 77 77 Telegram`

### Поиск ретушеров

```
ретушь фото памятники {город}
восстановление фото гравировка {город}
подрядчик гравировка памятники {город}
```

## Workflow (3 этапа)

### Этап 1: Сбор базы

1. **JSprav** — основной источник (до 100+ компаний)
2. **2GIS** — карты, отзывы
3. **Yell** — каталог
4. **Авито** — частные мастера

### Этап 2: Обогащение Telegram

Для каждого номера: `"{номер}" Telegram`

### Этап 3: Приоритизация

- С Telegram → приоритет для работы
- Без Telegram → в работе
