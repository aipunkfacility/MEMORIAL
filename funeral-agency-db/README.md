# Funeral Agency Database Agent

## Overview

Агент для сбора базы ритуальных агентств и производителей памятников по городам России. Используется для создания базы для рассылки и поиска подрядчиков (ретушеров).

## Приоритет контактов

**Telegram > WhatsApp > Email > Телефон**

Для каждой компании обязательно искать и записывать полные ссылки:
- Telegram: `https://t.me/username` или `https://t.me/username`
- WhatsApp: `https://wa.me/номер` (формат: wa.me/79xxxxxxxxx)

## Что собирать

### Обязательно для каждой компании
- Название организации
- Телефон (в формате +7XXX...)
- Адрес
- Сайт

### Контакты (ОЧЕНЬ ВАЖНО!)
- **Telegram** — ссылка t.me/username
- **WhatsApp** — ссылка wa.me/79xxxxxxxxx  
- **Email** — официальный
- **VK** — ссылка на группу

### Дополнительно
- Режим работы
- Особенности (производство, сроки, доставка)
- Упоминание ретуши/обработки фото
- Станки (лазерные, ударные)

## Поиск аккаунтов по номеру (ОБЯЗАТЕЛЬНО!)

Для КАЖДОГО номера делать поиск:
```
"{номер}" Telegram
"{номер}" WhatsApp
```

Примеры:
- `+7 903 955 81 17 Telegram`
- `8 937 821 77 77 WhatsApp`

После нахождения — записывать ПОЛНУЮ ссылку:
- Telegram: t.me/evlitos
- WhatsApp: wa.me/79039558117

## Процесс работы

1. Человек называет город
2. Агент создаёт папку `cities/{city}/`
3. Собирает данные в `data.md`
4. Ищет Telegram/WhatsApp по номерам
5. Обновляет ссылки в формате t.me, wa.me
6. Создаёт `summary.md` и `report.md`
7. Показывает результат
8. Ждёт подтверждения для следующего города

## Важно

- **Приоритет — Telegram**: всегда искать и записывать полную ссылку t.me
- **Полные ссылки**: wa.me/номер, t.me/username
- **Не придумывать**: только факты из источников
- **Проверять ссылки**: если не работают — указать
