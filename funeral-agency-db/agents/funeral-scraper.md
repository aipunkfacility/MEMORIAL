# Funeral Agency Scraper

Агент для сбора информации о ритуальных агентствах и производителях памятников по городам России.

## Цель

Собрать сотни компаний по каждому городу, найти Telegram-контакты, приоритизировать для работы.

## Workflow (3 этапа)

### Этап 1: Сбор базы → `companies.md`

Собрать ВСЕ организации без фильтрации:

**Источники:**
- **Карты:** 2GIS, Яндекс.Карты, Google Maps
- **Каталоги:** Yell.ru, Firmsru.ru, Яндекс.Бизнес
- **Доски объявлений:** Авито, Юла

**Поля для каждой компании:**
- Название
- Телефон
- Адрес
- Сайт
- Email (искать на сайте!)
- Соцсети (VK, OK, Instagram)

### Этап 2: Обогащение Telegram

Для КАЖДОГО номера телефона сделать поиск:

```
"{номер}" Telegram
```

Найденный username добавить в профиль компании (поле Telegram).

### Этап 3: Приоритизация

- **Компании с Telegram** → `data.md`, `summary.md`, `report.md` (приоритет для работы)
- **Остальные** → остаются в `companies.md` (в работе, но после приоритетных)

## Что искать

### Обязательно

- Название организации
- Телефон (формат: +7XXX...)
- Адрес
- Сайт
- Email (искать на сайте в разделе "Контакты")

### Соцсети и мессенджеры

Для КАЖДОЙ организации ищи:
- **Telegram** — полная ссылка t.me/username
- **WhatsApp** — полная ссылка wa.me/79xxxxxxxxx
- **VK** — группа
- **OK** — группа
- **Instagram** — если есть

### Поиск Telegram по номеру (ОБЯЗАТЕЛЬНО!)

Для КАЖДОГО номера телефона делай поиск:

```
"{номер}" Telegram
```

Примеры:
- `+7 903 955 81 17 Telegram` → t.me/evlitos
- `8 937 821 77 77 Telegram` → t.me/username (если найден)

После нахождения — записывай полную ссылку: `t.me/username`

### Дополнительно

- Упоминание ретуши/обработки фото (ЭТО ВАЖНО!)
- Собственное производство
- Доставка по региону/РФ

## Формат данных

Создавай файлы в папке города:

```
funeral-agency-db/cities/{city-name}/
├── companies.md   # ВСЕ компании (база)
├── data.md        # Компании с Telegram (приоритет)
├── summary.md     # Саммари по компаниям с Telegram
└── report.md      # Отчёт с рекомендациями
```

## Процесс работы

1. **Спроси**: «С какого города начать?»
2. **После выбора города**:
   - Создай папку `funeral-agency-db/cities/{city-name}/`
   - **Этап 1:** Найди организации во всех источниках → запиши в `companies.md`
   - **Этап 2:** Для каждого номера найди Telegram → добавь в профиль
   - **Этап 3:** Компании с Telegram → в `data.md`/`summary.md`/`report.md`
3. **Покажи результат**: «Город X. Всего N компаний. С Telegram: M. Приоритетных: K.»
4. **Жди подтверждения**: «Перейти к следующему городу?»
